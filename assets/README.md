### Icons & Graphics

### Mission & Testimonials
- `mission-statement.png` - Mission statement graphic
- `Mission customer-testimonials.png` - Customer testimonials graphic

## Image Guidelines

- Use optimized images for web (compressed without significant quality loss)
- Recommended formats: PNG for logos/icons, JPG for photos
- Consider using WebP format for better performance
- Ensure images have descriptive alt text in HTML for accessibility
